Learn how to install SAP Digital Payments Add-on (DPA).## Installation

Follow these steps to install the SAP DPA:

1. Install and configure the SAP DPA (directly from SAP).
2. [Onboard the DPA tenant](https://us-prod-dpa-doc.sap.stripeconnectors.com/docs/StripeSAPDPAAdapterUserGuide.pdf)on the Stripe Connector.
